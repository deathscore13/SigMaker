# SigMaker
### Универсальный генератор сигнатур для IDA 9.0+<br><br>

Данный плагин умеет:
* Создавать уникальные сигнатуры функций
* Создавать уникальные сигнатуры на основе позиции
* Создавать сигнатуры из выделенной области
* Проверять валидность сигнатуры
* Конвертировать паттерны IDA <-> Code

<br><br>
## Установка
1. Загрузить собранную версию из **[Releases](https://github.com/deathscore13/SigMaker/releases)**;
2. Переместить **`sigmaker.dll`** в **`корневая_папка_IDA\plugins\`**;
3. Запустить IDA и открыть `Edit --> Plugins --> SigMaker`, либо нажать `Ctrl+Alt+S`
